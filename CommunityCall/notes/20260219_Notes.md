# PowerShell OpenSSH Community Call Notes - February 19th, 2026

## Meeting notes

**PowerShell 7.6 LTS Release Update and Lifecycle Alignment:** Jason provided a comprehensive update on the status of the PowerShell 7.6 LTS release, addressing the reasons for its delay, the current plan for the release candidate and general availability, and ongoing community concerns about lifecycle alignment with .NET releases, inviting further feedback via the PowerShell repository discussion.

**2026 PowerShell, OpenSSH, and DSC Team Investments and Roadmap:** Steve presented the team's planned investments for 2026, covering security and compliance priorities, feature work for PowerShell 7.7, improvements to PSResourceGet and PowerShell Gallery, OpenSSH enhancements, and the Desired State Configuration (DSC) roadmap, while inviting community feedback on several proposals.

**PSResourceGet and PowerShell Gallery Updates:** Anam provided updates on the PSResourceGet 1.20 RC3 release, which resolves a dependency issue with the AZ module, and addressed ongoing intermittent search indexing problems in the PowerShell Gallery, offering guidance and requesting user feedback.

**PowerShell Documentation Maintenance and Updates:** Sean summarized recent maintenance work on PowerShell documentation, highlighting a major rewrite of the PowerShellGet install documentation to clarify both online and offline installation scenarios, and improvements to guidance for bootstrapping modules on new Windows systems.

**DSC and Bicep Integration Demonstration:** Speaker 1 demonstrated the integration of Desired State Configuration (DSC) v3 with Bicep, showcasing authoring and orchestrating DSC resources using Bicep modules, type resolution, and advanced orchestration features such as parameterization, loops, and dependency management.

**PowerShell Web REPL and Static Site Generation Performance:** James presented a web-safe REPL for PowerShell called Reptile, designed for educational and secure interactive use, and shared benchmarking results demonstrating PowerShell's superior performance in static site generation compared to other tools.

**PowerShell Gallery Migration to AKS Update:** Amber provided an update on the ongoing migration of the PowerShell Gallery to Azure Kubernetes Service (AKS), noting that the project remains a top priority, is progressing through testing phases, and aims to improve scaling and reliability without disrupting users.
