# PowerShell OpenSSH Community Call Notes - March 19th, 2026

## Meeting notes

**MAR and PSResourceGet Updates:** Sydney provided a comprehensive update on the adoption of the .NET ORAS library for OCI registry support in PSResourceGet 1.3, discussed the migration of Microsoft 1P packages to MAR, and addressed community questions from Sean Williams (SIE) and Anam Navied regarding migration timelines, dual publishing, and technical limitations.

**Remote Azure Management MCP Server Private Preview:** Sydney announced the limited private preview of the remote Azure Management MCP server, requesting community assistance with authentication testing and explaining the current capabilities and onboarding process.

**PowerShell 7.6 Release and Support Updates:** Aditya Patwardhan and Steve Lee (POWERSHELL) confirmed the general availability of PowerShell 7.6, discussed ongoing package propagation, and addressed enterprise support concerns and plans for improving future .NET alignment.

**DSC v3 Preview 13 Features and Demo:** Steve Lee (POWERSHELL) demonstrated new features in DSC v3 Preview 13, including a new service resource, optional feature list support, and discussed experimental resource feedback and schema validation with input from Andres Bohren and Aditya Patwardhan.

**PowerShell Update Notification Community Discussion:** Jason Helmick invited the community to participate in a focused discussion about the future of the PowerShell update notification, seeking input on whether to keep or remove the feature.

**Homebrew Notarization and Community Formula:** Jason Helmick updated the group on ongoing efforts to support notarization for PowerShell in Homebrew, highlighting a community-contributed formula by Gunny Bush as an alternative for Mac users.

**Linux Distribution Support and Release Info Tooling:** Jason Helmick and Sean Wheeler discussed the status of RPM packages for RHEL 10 and Debian 13, announced ongoing work on Deb ARM support, and demonstrated Sean Wheeler's Release Info module for tracking OS and PowerShell release lifecycles.

**Documentation Updates and Homebrew Installation Guidance:** Sean Wheeler described recent refactoring of PowerShell setup documentation, including moving Homebrew installation instructions to a dedicated section as Homebrew is now a community-supported solution.

**DSC v3 LCM and Pull Server Replacement Demo:** Thomas Nieto presented a high-level demo of a custom LCM and pull server replacement for DSC v3, featuring a dashboard for node compliance, configuration management, parameter scoping, and composite configurations.
